# Data_Analysis
This is my first Data Analytics project focused on marketing performance analysis using Microsoft Excel. In this project, I analyzed sales data of different sales executives across regions to evaluate performance, target achievement percentage, and overall business growth
The dataset includes daily sales, total sales, targets, and performance metrics. Using Excel tools like Pivot Tables, Charts, Conditional Formatting, and Formulas, I derived insights to help understand:

Sales performance by region

Target achievement percentage

Top and low-performing sales executives

Gap analysis (Away from Target %)

Weekly sales trends

Tools & Skills Used:

Microsoft Excel

Pivot Tables

Data Cleaning

Percentage & Statistical Calculations

Data Visualization (Charts & Graphs)

Business Insight Generation
